# Habit Tracker: Basado en "Hábitos Atómicos"

Este proyecto es una solución digital diseñada para ayudar a las personas a construir hábitos positivos o eliminar los negativos, aplicando la filosofía de James Clear en su libro *Hábitos Atómicos*. 

La aplicación se enfoca en la consistencia diaria: si dejas pasar un día, el progreso se reinicia, fomentando la disciplina necesaria para alcanzar la meta de los 66 días.

 Características principales

* **Gestión de Hábitos:** Crear, editar y visualizar tus hábitos diarios.
* **Sistema de Rachas:** Si no marcas un hábito como completado, el contador vuelve a cero automáticamente.
* **Visualización de Progreso:** Barra de progreso dinámica que cambia de **rojo** (inicio) a **verde** (meta cumplida) según te acerques a los 66 días.
* **Autenticación:** Creación de cuenta e inicio de sesión seguro.

# Stack Tecnológico

Este proyecto utiliza el stack **MERN** (con variaciones modernas):

* **Frontend:** [Next.js](https://nextjs.org/) (React Framework)
* **Backend:** [Express.js](https://expressjs.com/) (Node.js)
* **Base de Datos:** [MongoDB](https://www.mongodb.com/) (NoSQL)
